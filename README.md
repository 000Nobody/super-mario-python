# Super Mario Implementation in Python

This is inspired by Meth-Meth-Method's: https://github.com/meth-meth-method/super-mario/

## Running

* Install Python
* Run `pip install pygame`.
* Clone repository.
* Run `python main.py`.

## Dependencies

* Pygame

## Have Fun

Have fun by pressing left or right mouse button ;)
